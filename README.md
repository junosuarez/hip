# hip
minimal http server

preview, experimental, work in progress, etc, etc

## usage

    var hip = require('./')

    hip(function(h) {
      return 'hello thar'
    }).listen(1080)




## installation

    $ npm install hip


## running the tests

From package root:

    $ npm install
    $ npm test
